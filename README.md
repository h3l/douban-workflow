# douban-workflow

因为网上找到的大多数豆瓣的 workflow 都因为豆瓣 api 的关闭而失效，所以，自己动手，丰衣足食。

支持搜索书籍、电影和音乐。

`dbb` (douban book 简称)
`dbm`(douban movie 简称)
`dby`(douban yinyue简称，与dbm冲突，折衷)

### 安装

* [点击 douban.alfredworkflow 下载安装](https://github.com/h3l/douban-workflow/raw/master/douban.alfredworkflow)
